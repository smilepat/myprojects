## 3. 사용자 결정 대기 4건 — 권장안 포함

LAUNCH_PLAN.md §11의 결정 사항. 컨소시엄 결성 단계에서 답해야 함.

### 결정 1: 글로벌 출시 시점

| 옵션 | 장점 | 단점 |
|------|------|------|
| **v1 한국만** ⭐ 권장 | 단일 컴플라이언스 (KISA만), 빠른 출시, 컨소시엄 합의 쉬움 | 글로벌 EFL 시장 진입 늦음 |
| v1.5 한국+일본/미국 | LTV ↑, Apple 글로벌 채널 활용 | COPPA·GDPR 추가 부담, 비용 ↑ |

**권장**: v1 한국만. KERIS-Samsung 5개국 자산은 글로벌 진입 시점에 활용. Phase 3 (2027 Q3)부터 일본 검토.

### 결정 2: 학습자 연령 하한

| 옵션 | 장점 | 단점 |
|------|------|------|
| **5세 (파닉스 Phase 1)** ⭐ 권장 | 12년 trajectory 완전체, 학부모 유입 빠름 | iOS Kids Category 압력, COPPA 압력 |
| 7세 (사이트워드부터) | 컴플라이언스 부담 적음 | "파닉스부터 대입까지" 메시지 약화 |

**권장**: 5세 — 단, **Kids Category는 회피**하고 일반 12+ 등급으로 출시. 부모 게이트 강제 + 음성 데이터 즉시 삭제. 컴플라이언스 부담은 PG급 시스템으로 흡수 가능.

### 결정 3: 수능 기출 라이선스

| 옵션 | 장점 | 단점 |
|------|------|------|
| **csat-item-factory 자체 변형 우선** ⭐ 권장 | 라이선스 협상 부담 0, 저작권 안전 | "정확히 그 문제" 매력 약함 |
| KICE 라이선스 협상 | 권위 확보 | 협상 6–12개월, 비용 불명 |

**권장**: v1은 자체 변형. KICE 협상은 **Phase 2** (강원도 광역 배포 후 사회적 증거 확보 시점에) 시작. 컨소시엄 명의로 협상하면 단독보다 협상력 N배.

### 결정 4: 베타 모집 채널

| 옵션 | 장점 | 단점 |
|------|------|------|
| **강원도 교사 + 학부모 둘 다** ⭐ 권장 | B2B 신뢰 + B2C 사용량 동시 확보 | 모집 분산 |
| 강원도 교사만 | 신뢰 ↑, B2B 진입 빠름 | 사용량 부족 |
| 학부모만 | 사용량 ↑ | B2B 진입 느림 |

**권장**: 둘 다. 교사 50 + 학부모 150 = 200명 ([LAUNCH_PLAN.md §출시 후 운영](LAUNCH_PLAN.md)).

## 5. Pretendard 설치 1-liner

### Windows (PowerShell — 관리자 권한)

```powershell
# 한 번에 다운로드 + 설치 (Windows용 .ttf 7개 파일)
$ver = "v1.3.9"
$url = "https://github.com/orioncactus/pretendard/releases/download/$ver/Pretendard-$ver.zip"
$tmp = "$env:TEMP\pretendard.zip"
$out = "$env:TEMP\pretendard"
Invoke-WebRequest $url -OutFile $tmp
Expand-Archive $tmp -DestinationPath $out -Force
Get-ChildItem "$out\public\static\*.otf" | ForEach-Object { Copy-Item $_.FullName "$env:WINDIR\Fonts\" -Force }
```

### 또는 GUI

1. [pretendard.org](https://github.com/orioncactus/pretendard/releases) → 최신 zip 다운로드
2. `public/static/` 폴더 안 `.otf` 파일 전체 선택
3. 우클릭 → **모든 사용자용 설치**
4. VS Code 재시작 → Marp 미리보기에서 폰트 정확히 적용

> **참고**: 슬라이드 CSS는 이미 Pretendard Variable CDN을 import하므로, 설치 안 해도 브라우저 뷰어는 정상 작동. 로컬 설치는 **VS Code 내 Marp 미리보기 + PPTX 내보내기** 시점에만 필요.
